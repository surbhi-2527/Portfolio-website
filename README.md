# Portfolio-website

Welcome to my personal portfolio website!
This project showcases my work, skills, and projects as a Computer Science student specializing in Artificial Intelligence.

---

## ✨ About the Project

This is a **modern, responsive portfolio website** built using React.
It highlights my projects, technical skills, and provides a way to connect with me.

The design focuses on a **soft, warm, and minimal aesthetic** to create a pleasant user experience.

## 🚀 Features

* 🌐 Responsive design (works on all devices)
* 🎯 Clean and modern UI
* 🧩 Component-based architecture (React)
* 📁 Projects showcase section
* 📬 Contact form
* 🌸 Soft and warm theme
